Projects

  1. Dynamically Optimized Cycling Navigation with Real-Time Adaptive Routing

    This study develops a real-time cycling route optimization model addressing poor infrastructure, disconnected routes, and traffic. Using Monte Carlo rollouts, OpenStreetMap (OSM), OSMnx, and NetworkX, the model dynamically adjusts road weights based on safety, traffic, and real-time obstacles (1% random blockage). It integrates Dijkstra’s/A for fallback routing and Strongly Connected Components (SCC) to ensure connectivity. The model continuously optimizes safer, cycle-friendly routes, validated through Python-based visualization.

  2. Early Stage Lung Cancer Detection Using Deep Convolutional Neural Networks: A Comprehensive AI-Driven Approach for Precision Pulmonary Nodule Classification and Enhanced CT Imaging

    Developed a CNN model using the LIDC-IDRI dataset to detect lung nodules in CT scans, improving early diagnosis of lung cancer. Leveraged Google Cloud for scalable data storage and Google Colab for GPU-accelerated model training. Integrated explainability tools to assist radiologists in making informed diagnoses. Focused on creating a scalable, real-time AI solution for clinical deployment.

  3. Microphone Array-Based Direction of Arrival for Gunshot Detection (Nominee for Smart India Hackathon 2024)

    Developed a real-time gunshot detection system using a 6-microphone array to capture sound and determine the direction of arrival (DoA) using Time Difference of Arrival (TDoA) algorithms. Leveraged FPGA-based processing, bandpass filtering, and sound classification algorithms (KNN, STFT) for noise reduction and accurate detection. The system provides visual feedback and enhances situational awareness for military and security applications, reducing false positives and enabling rapid response.
